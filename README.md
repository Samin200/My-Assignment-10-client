# MovieMaster Pro

Live Site: [https://your-live-site-url.com](https://your-live-site-url.com)

---

## Features
- User authentication via Email/Password and Google
- Add, view, and manage your personal movie collection
- Filter movies by genre, rating, and search
- Create and manage a personal watchlist
- Update user profile with name and photo
- Like movies and leave reviews/comments
- Responsive design for mobile and desktop

---

## Tech Stack
- Frontend: React, TailwindCSS, Axios
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: Firebase Auth
